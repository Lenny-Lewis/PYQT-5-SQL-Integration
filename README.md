🎓 Student Information Management System
Overview

The Student Information Management System is a desktop-based application developed using PyQt5 and SQL Server. It provides an intuitive graphical interface for managing student records with full CRUD (Create, Read, Update, Delete) functionality.

This system is designed to simplify student data management while demonstrating integration between a Python GUI and a relational database.
